# Minervini 趨勢模板選股工具 - 台股版

## 專案簡介

本工具是以傳奇交易者 Mark Minervini 的「第二階段趨勢模板」(Stage 2 Trend Template) 為核心的台灣股票篩選與分析系統。它結合技術面指標與基本面數據，幫助投資者快速識別符合趨勢模板條件的潛力股。

直接在瀏覽器中開啟 HTML 檔案即可使用，無需安裝任何軟體依賴。

---

## 功能特色

✅ **Minervini 趨勢模板評分** - 自動檢查 6 項技術面條件 + 2 項基本面指標，產生 0-100 分綜合評分

✅ **K 線圖與均線疊圖** - 動態呈現最近 250 個交易日的 K 線、成交量，及 50/150/200 日均線

✅ **均線多頭排列驗證** - 收盤價 > 50MA > 150MA > 200MA，確認股票處於強勢上升趨勢

✅ **基本面數據自動抓取** - 透過 FinMind API 即時擷取月營收年增率、EPS 成長等財務指標

✅ **智能交易規劃（三種模式）** - 穩健型（5% 停損）、積極型（8% 停損）、趨勢跟蹤型（移動停利追蹤大波段）

✅ **觀察名單追蹤** - 使用 IndexedDB 持久化儲存關注股票，支援批量更新價格、匯出/匯入 JSON、重置名單

✅ **轉折點偵測** - 即時監測 MA 交叉、MACD 金死叉、趨勢危險信號，提前預警

✅ **成本基準出場** - 支援輸入實際持倉成本，設定以成本為基準的分層出場計畫

## 使用方法

### 基本操作

1. **開啟工具**
   - 在瀏覽器中直接開啟 `minervini-screener.html` 檔案
   
2. **輸入股票代碼**
   - 在「台股代號」欄位輸入 4-6 位數字（例如：2330）
   - （選填）在「我的持倉成本」輸入買入價格，用於計算出場計畫
   - 按 Enter 或點擊「開始分析」按鈕

3. **查看分析結果**
   - **評分卡** - 顯示該股是否符合 Minervini 模板
   - **檢查項目** - 逐項檢視 6 項技術面 + 2 項基本面條件
   - **轉折點偵測** - 顯示當前的 MA 交叉、MACD、趨勢訊號
   - **K 線圖** - 近 250 日走勢與均線疊圖
   - **交易計畫** - 根據風險承受度選擇保守/激進方案，包含進場、目標、停損設定
   - **觀察名單** - 加入追蹤後可一鍵更新所有觀察股票的價格

### 進階功能

**FinMind API Token（選填）**
- 免費註冊 [finmindtrade.com](https://finmindtrade.com/) 申請 token
- 不填也能使用，但 API 額度有限（300 次/小時）
- 填入 token 可提升至 600 次/小時

**手動覆蓋基本面資料**
- 若自動抓取失敗，可在此欄位手動輸入：
  - 月營收年增率 (%)
  - 當季 EPS (元)
  - 去年同期 EPS (元)

**觀察名單**
- 分析完股票後，點擊「加入觀察」將其儲存至本地 IndexedDB
- 展開觀察名單，點擊「一鍵更新價格」重新取得所有關注股票的最新價格與漲跌
- 點擊「移除」刪除不再追蹤的股票
- 「匯出 JSON」— 將觀察名單下載為 JSON 檔案備份
- 「匯入 JSON」— 從 JSON 檔案還原，支援合併或覆蓋模式
- 「重置名單」— 清空所有資料（有二次確認防誤觸）

## Minervini 趨勢模板原理

### 核心概念

Mark Minervini 的「Stage 2」選股模板是一套基於技術面與基本面結合的趨勢交易系統，強調在股票開始上升趨勢的早期階段進場。

### 六項技術面條件（必須）

1. **多頭排列** - 收盤價 > 50MA > 150MA > 200MA
   - 確保股價在短期、中期、長期都處於上升狀態

2. **底部區域** - 股價高於 52 週低點 30% 以上
   - 避免在底部持有太久的股票

3. **接近高點** - 股價距 52 週高點不超過 25%
   - 表示股票正在創新高，具備動能

4. **成交量充足** - 日均成交量 > 1,500 張
   - 確保流動性，便於進出場

5. **營收動能** - 月營收年增率 > 20%（基本面）
   - 業績增長是股價上升的基本支撐

6. **EPS 正成長** - 當季 EPS > 0 且超越去年同期（基本面）
   - 獲利成長驗證營收動能的真實性

### 計分方法

- 技術面 4 項條件：滿分 60 分（每項 15 分）
- 基本面 2 項條件：滿分 40 分（每項 20 分）
- **80+ 分：強力推薦關注** — 符合絕大多數條件，進場訊號明確
- **50-79 分：部分符合** — 有潛力但尚需觀察，持續追蹤
- **<50 分：不符合模板** — 當前不滿足關鍵條件

### 轉折點偵測

工具會自動監測以下關鍵訊號，幫助投資者及早發現趨勢變化：

- **均線排列狀態** - 多頭排列 / 空頭排列 / 均線糾結
- **黃金交叉 / 死亡交叉** - 50MA 與 150MA 的穿越
- **價格 vs 50MA** - 站穩之上（偏多）或跌破（轉弱）
- **均線斜率** - 50MA 近 10 日上揚或轉弱
- **價格 vs 200MA** - 長期趨勢完整或破位
- **52 週位置** - 接近高點（強勢）/ 區間中段 / 接近低點（弱勢）
- **MACD 金叉 / 死叉 / 正向 / 負向** - 動量指標的完整狀態

## 交易紀律與風險管理

### Minervini 核心紀律

1. **絕不向下攤平** — 股價跌到停損就走，不找藉口加碼
2. **單筆風險控制** — 每筆交易虧損不超過總資金的 1-2%
3. **進場需要理由，出場不需要** — 停損觸發就是出場理由
4. **前 2-3 天是關鍵** — 進場後若股價沒有如預期上漲，考慮主動減碼

### 三種出場策略

工具自動產生三套出場方案：

**穩健型（停損 5%）**
- T1 停利 +10%（賣 1/3）→ T2 +15%（賣 1/3）→ T3 +25%（賣 1/3）
- 移動停利：股價跌破 50MA 或 150MA 時出清

**積極型（停損 8%）**
- T1 停利 +16%（賣 1/3）→ T2 +24%（賣 1/3）→ T3 +40%（賣 1/3）
- 承受更大波動，換取更高報酬空間

**趨勢跟蹤型（移動停利）**
- 適合強勢趨勢股，用均線追蹤取代固定停利，目標抱住大波段
- 四階段自動升級：
  - +0~15%：固定停損（成本 -8%）
  - +15~30%：停損上移至成本價，用 50MA 追蹤
  - +30~50%：賣 1/4 鎖利，50MA 或最高回撤 15% 取高者
  - +50~100%：飆股模式，只看週收盤價跌破 10 週均線才出場
  - +100% 以上：翻倍模式，連續兩週收盤低於 10 週均線才出清
- 關鍵原則：用週收盤價判斷，不被日內波動洗出去

### 持股張數計算

假設總資金 100 萬元，最大單筆虧損 1%（= 1 萬元）：

停損幅度 = 進場價 × 5% ~ 8%

可買張數 = 10,000 ÷ 停損幅度 ÷ 1,000

例如進場價 100 元，停損幅度 5 元，則可買 2 張

## 局限性與免責聲明

### 工具局限

1. **無法保證獲利** - 股市有風險，過去績效不代表未來表現
2. **資料延遲** - FinMind API 可能有 1-2 日的資料延遲，不適合超短期交易
3. **基本面資料可能缺失** - 部分新上市股票或小型股可能無法取得完整 EPS 資料
4. **網路依賴** - 需要穩定網路連線以取得實時資料

### 免責聲明

本工具僅供學習與研究之用，任何投資決策應基於投資者自身的風險承受度與長期目標。使用本工具產生的任何損失，與開發者無關。

**重要提示：**
- 股票投資涉及本金損失風險
- 任何投資決策前應諮詢專業投資顧問
- 嚴格遵循停損紀律，不向下攤平
- 建立完整的交易計畫與資金管理制度
